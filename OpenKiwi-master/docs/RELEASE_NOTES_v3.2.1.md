## OpenKiwi 3.2.1

预编译调试包 **OpenKiwi321.apk**（便于快速体验；生产环境请自行 release 签名构建）。

### 安装

1. 下载附件 `OpenKiwi321.apk`
2. 允许「安装未知来源应用」
3. 安装后：设置 → 模型配置 → 填入你的 OpenAI 兼容 API

### 本版亮点（相对早期版本）

- **聊天顶栏 / 输入栏**：适配刘海与挖孔安全区，拉开按钮间距，避免重叠
- **飞书**：支持手机端直连长连接（可选）+ PC 伴侣转发
- **PC 伴侣**：WebSocket 文件上传/下载、mDNS 发现手机（`_openkiwi._tcp`）、与手机端流式协议对齐（`chat_stream` / `chat_end`）
- **定时任务**：WorkManager + Room，侧边栏「定时」入口
- **剪贴板监听**：可选快捷操作（设置中开启）
- **通知自动回复**：RemoteInput 直回（按应用规则）
- **微信/QQ**：`app_reply_bot` 等工具链
- **桌面小组件**：快捷发起对话
- **本地 RAG**：文档/下载目录索引 + `local_rag` 工具；设置中「重建索引」
- **自动化配方**：GuiAgent 步骤流 + `run_recipe` 工具 + 配方页
- **语音唤醒**：设置中可配置唤醒词与开关

### 系统要求

- Android 8.0（API 26）及以上
- 需自备大模型 API；部分能力依赖无障碍、通知监听等权限
