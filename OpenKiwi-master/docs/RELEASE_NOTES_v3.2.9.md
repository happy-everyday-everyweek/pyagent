## OpenKiwi v3.2.9

增量构建，在 v3.2.8 能力基础上包含近期修复与提示词优化。

### 变更摘要
- **内嵌 Python 环境说明**：向模型明确 Chaquopy 下可用标准库与 pip 包（`requests`、`beautifulsoup4`），避免误用不存在的包。
- **通知分析时间**：通知 LLM 分析使用**每次请求时**的当前时间，便于正确解析「明天」等相对时间。
- **Agent 系统提示**：主对话系统提示附带实时时间与时区，便于日程、日历类推理。
