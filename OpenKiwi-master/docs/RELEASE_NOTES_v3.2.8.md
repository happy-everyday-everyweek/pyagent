## OpenKiwi v3.2.8

### New Features
- **MCP Protocol**: SSE/STDIO transport, MCP settings UI, auto tool bridging
- **Multi-LLM Provider**: OpenAI / Anthropic / Gemini unified abstraction
- **Plugin SDK**: Dynamic plugin loading with manifest system
- **Agent Engine V2**: Planning, reflection, sub-agent management
- **Persistent Terminal**: Shell sessions survive across tool calls

### UI Overhaul
- Paper/magazine-style theme (warm neutral palette, Serif font, blue accent)
- Drawer redesign: conversations prioritized, tools in collapsible grid
- Feature page back-navigation auto-reopens drawer
- File attachments display as compact cards instead of raw content
- Unified styling across overlays, terminal, settings
