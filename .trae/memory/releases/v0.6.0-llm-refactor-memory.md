# v0.6.0 LLM模块重构任务记忆

## 任务概述

完成PyAgent v0.6.0版本的核心重构，包括LLM模块重构、设备ID系统、统一工具调用接口、Mate模式简化、Web UI重构和ClawHub集成。

## 完成路径

### 1. LLM模块核心重构

**文件修改**:
- `src/llm/types.py` - 新增ModelTier、VerticalType枚举，更新ModelConfig和VerticalModelConfig
- `src/llm/config.py` - 完全重构配置加载器，支持新格式
- `src/llm/client.py` - 重构客户端，支持分级模型和垂类模型
- `config/models.yaml` - 更新配置格式

**关键实现**:
- 基础模型（必填）作为默认模型
- 分级模型：STRONG/PERFORMANCE/COST_EFFECTIVE
- 垂类模型：SCREEN_OPERATION/MULTIMODAL/CUSTOM
- 任务类型到模型的自动映射
- 多模态模型回退机制

### 2. 设备ID系统

**新增文件**:
- `src/device/__init__.py`
- `src/device/device_id.py`

**关键实现**:
- 设备ID格式：日期+10位随机数 → SHA256哈希前16位
- 持久化存储到 `data/device/device_id.json`
- 单例模式管理

### 3. 统一工具调用接口

**新增文件**:
- `src/tools/__init__.py`
- `src/tools/base.py` - ToolLifecycle、UnifiedTool基类
- `src/tools/registry.py` - ToolRegistry工具注册中心
- `src/tools/skill_tool.py` - Skill工具包装器
- `src/tools/mcp_tool.py` - MCP工具包装器

**关键实现**:
- 三阶段调用：激活 → 调用 → 休眠
- 统一的工具状态管理
- 工具列表包含设备ID

### 4. Mate模式重构

**文件修改**:
- `src/todo/mate_mode.py` - 移除推理可视化，简化为协作模式开关
- `src/todo/types.py` - 简化MateModeState
- `src/web/routes/todo_routes.py` - 移除不需要的API端点

### 5. Web UI重构

**文件修改**:
- `frontend/src/views/ChatView.vue` - 斜杠命令、快捷菜单
- `frontend/src/views/TasksView.vue` - 任务进度卡片重构
- `frontend/src/views/ConfigView.vue` - 模型配置界面优化

### 6. ClawHub集成

**新增文件**:
- `src/skills/clawhub.py` - ClawHub Skill安装器

**关键实现**:
- URL解析：`clawhub://skill-name` 或 `https://clawhub.io/skills/...`
- Skill技能安装和卸载
- 自动创建SKILL.md文件和目录结构

### 7. 文档更新

**文件修改**:
- `CHANGELOG.md` - 添加v0.6.0版本记录
- `AGENTS.md` - 更新版本信息和核心特性
