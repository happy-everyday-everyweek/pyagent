## 任务概述

优化移动端模块，将手机操作封装为单一工具 `phone_operation`，便于 AI 理解和调用。增强 AutoGLM 子代理实现，支持循环执行模式。

## 技术亮点

1. **统一工具接口**: 将复杂的手机操作封装为单一工具，简化 AI 调用
2. **循环执行模式**: 截图 -> 分析 -> 规划 -> 执行 -> 验证
3. **多模态集成**: 使用 screen-operation 垂类模型分析屏幕
4. **失败恢复**: 支持操作失败后自动重试
5. **虚拟屏幕**: 支持在虚拟屏幕执行操作，不影响主屏幕

---

## 可优化方向

1. **UI 元素定位**: 当前仅支持坐标操作，可添加元素级定位（参考 Operit 的 UINode）
2. **无障碍服务**: 集成 AccessibilityService 获取 UI 树
3. **操作录制**: 支持录制操作序列并回放
4. **跨应用流程**: 支持跨多个应用的复杂操作流程
5. **性能优化**: 减少截图和分析的延迟
