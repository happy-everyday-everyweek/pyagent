## 功能清单

### XlsxCreator - 工作簿创建器

- 创建空白工作簿
- 从模板创建工作簿
- 添加工作表
- 设置单元格值（字符串、数字、布尔值）
- 设置公式

### XlsxEditor - 工作簿编辑器

- 列出工作表
- 获取单元格值
- 获取范围数据
- 设置单元格值
- 查找单元格

### SheetManager - 工作表管理器

- 列出工作表信息
- 添加工作表
- 删除工作表
- 重命名工作表
- 移动工作表位置
- 复制工作表

### ChartManager - 图表管理器

- 添加图表（柱状图、折线图、饼图、散点图、面积图）
- 设置图表标题、坐标轴标题
- 设置图表位置和大小
- 列出图表
- 删除图表

## 技术要点

1. **OOXML 结构**: 遵循 Excel 的 OOXML 结构，正确处理 workbook.xml、sheet.xml、关系文件
2. **单元格类型**: 支持字符串、数字、布尔值和公式四种类型
3. **图表支持**: 通过创建 chart.xml 和 drawing.xml 实现图表功能
4. **内存管理**: 使用临时目录处理，完成后自动清理
