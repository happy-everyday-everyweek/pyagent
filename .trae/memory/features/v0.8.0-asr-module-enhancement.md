### 4. 技术实现细节

#### 4.1 引擎自动检测
```python
def _detect_engine(self) -> str:
    """检测可用的ASR引擎"""
    engines = ["faster_whisper", "whisper", "dummy"]
    for engine in engines:
        try:
            __import__(engine.replace("_", "-"))
            return engine
        except ImportError:
            continue
```

#### 4.2 音频预处理流程
1. 保存音频数据到临时文件
2. 使用FFmpeg转换为标准格式（16kHz, mono, PCM）
3. 加载处理后的音频数据
4. 调用ASR引擎进行识别
5. 清理临时文件

#### 4.3 异步处理
- 使用 `asyncio.run_in_executor` 将同步的Whisper调用转换为异步
- FFmpeg处理也使用线程池执行
- 支持并发处理限制（通过环境变量 `PYAGENT_FFMPEG_MAX_CONCURRENCY`）

## 使用示例

### 基本使用
```python
from src.voice.asr import ASRModule, ASRConfig

# 创建配置
config = ASRConfig(
    language="zh-CN",
    whisper_model="base",
    engine="auto"
)

# 创建ASR模块
asr = ASRModule(config)
asr.initialize()

# 转录音频文件
result = await asr.transcribe_file("audio.mp3")
print(result)

# 转录音频数据
with open("audio.wav", "rb") as f:
    audio_data = f.read()
result = await asr.transcribe(audio_data)
print(result)
```

### 高级配置
```python
config = ASRConfig(
    language="en-US",
    whisper_model="medium",
    whisper_device="cuda",
    whisper_compute_type="float16",
    enable_vad=True,
    vad_threshold=0.6,
    use_ffmpeg=True
)

asr = ASRModule(config)
asr.initialize()

# 添加热词
asr.add_hotwords({"PyAgent": 2.0, "智能体": 1.5})

# 转录
result = await asr.transcribe_file("audio.mp3")
```
