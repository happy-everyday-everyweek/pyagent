### DeviceType 自动检测逻辑
```python
class DeviceType(Enum):
    PC = "pc"
    MOBILE = "mobile"
    SERVER = "server"
    EDGE = "edge"

    @classmethod
    def auto_detect(cls) -> "DeviceType":
        system = platform.system().lower()
        machine = platform.machine().lower()
        # Linux + ARM = EDGE 或 MOBILE
        # Windows/Darwin = PC
        # Linux x86 = SERVER
```
