- 添加了多引擎自动检测和降级机制
- dummy引擎作为最后的降级方案

### 3. PDF解析器完善

- 添加了基于pdfplumber的备用解析方案

- 修复了基础解析返回空文档的问题
- 完善了元数据读取功能

### 4. 金融智能体完善

- 集成了yfinance真实金融数据API
- 添加了缓存机制（TTL 300秒）
- 添加了RateLimiter限流机制（100次/分钟）
- 添加了错误处理和优雅降级

### 5. 前端修复
