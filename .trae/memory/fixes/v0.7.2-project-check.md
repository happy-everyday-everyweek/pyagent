- todo（AI原生Todo、Mate模式）
- llm（多模型支持、分级模型、垂类模型）

- skills（技能加载、解析、执行）
- device（设备ID、域管理、同步引擎）
- storage（分布式存储、文件追踪）
- human_tasks（人类任务管理、分类、通知）
- document（文档管理、ONLYOFFICE连接器）
- video（视频项目管理、编辑核心）
